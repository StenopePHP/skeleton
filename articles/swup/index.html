Swup

Swup is a library that allows you to give a SPA feel to your static website.

This skeleton integrates it using Symfony UX and configures scroll and progress plugins to enhance the navigation experience.

Note

A custom style is provided by the skeleton theme in assets/scss/skeleton-theme/components/swup.scss.
If you'd like to keep Swup, think about creating your own before removing the skeleton theme.
